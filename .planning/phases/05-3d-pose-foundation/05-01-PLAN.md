<feature>
  <name>Pose Utilities: Extraction, Frontalization, Normalization</name>
  <files>utils/pose_utils.py, tests/test_pose_utils.py</files>
  <behavior>
    POSE-01 (extract_pose_angles):
    - Identity matrix (eye(4)) -> pitch=0, yaw=0, roll=0 (within 1e-6)
    - Known 30-deg yaw rotation matrix -> yaw approx 30 degrees (within 1.0 deg tolerance)
    - Known 45-deg yaw rotation matrix -> yaw approx 45 degrees
    - Known 20-deg pitch rotation matrix -> pitch approx 20 degrees
    - Matrix with uniform scale factor 2.0 -> same angles as without scale
    - Returns dict with keys: pitch, yaw, roll (floats in degrees), matrix (raw 4x4 ndarray)

    POSE-02 (frontalize_landmarks):
    - Identity rotation + any landmarks -> output equals input (within 1e-10)
    - Landmarks rotated by known 30-deg yaw, then frontalized -> matches original within 2-3% IPD error
    - Landmarks rotated by known 45-deg yaw, then frontalized -> matches original within 3% IPD error
    - Combined pitch+yaw rotation -> frontalized matches original within tolerance

    POSE-03 (normalize_landmarks_3d):
    - Any landmarks -> output IPD (distance between indices 468 and 473) equals 1.0
    - Landmarks scaled by factor 2x -> normalized result identical to normalized original
    - Near-zero IPD (both iris at same point) -> returns copy unchanged, ipd=1.0
    - Returns tuple of (normalized_landmarks, original_ipd)

    Head dimensions (compute_head_dimensions):
    - Returns dict with width, height, depth in IPD-normalized units
    - Near-zero IPD -> uses ipd=1.0 fallback
  </behavior>
  <implementation>
    Create utils/pose_utils.py with:

    1. Constants: LEFT_IRIS_CENTER = 468, RIGHT_IRIS_CENTER = 473

    2. extract_pose_angles(transform_matrix: np.ndarray) -> dict:
       - Extract 3x3 rotation from top-left of 4x4 matrix
       - Remove uniform scale: divide by det(R)^(1/3) using np.cbrt(np.linalg.det())
       - Create SciPy Rotation via Rotation.from_matrix(rot_pure)
       - Decompose to Euler angles: r.as_euler('XYZ', degrees=True) -> pitch, yaw, roll
       - Return {'pitch': float, 'yaw': float, 'roll': float, 'matrix': transform_matrix}

    3. frontalize_landmarks(landmarks_3d: np.ndarray, transform_matrix: np.ndarray) -> np.ndarray:
       - Extract rotation (same scale removal as above)
       - Compute inverse rotation: r.inv()
       - Center landmarks around centroid (mean of all 478 points)
       - Apply inverse rotation: r_inv.apply(centered)
       - Re-add centroid
       - Return (478, 3) frontalized landmarks

    4. normalize_landmarks_3d(landmarks_3d: np.ndarray) -> tuple[np.ndarray, float]:
       - Compute 3D Euclidean IPD between indices 468 and 473
       - Guard: if ipd < 1e-8, return (landmarks.copy(), 1.0)
       - Compute midpoint between iris centers
       - Center on midpoint, divide by IPD
       - Return (normalized, ipd)

    5. compute_head_dimensions(landmarks_3d: np.ndarray, ipd: float) -> dict:
       - Compute bounding box extents (max - min) along each axis
       - Divide by IPD (guard ipd < 1e-8 -> use 1.0)
       - Return {'width': float, 'height': float, 'depth': float}

    Test fixtures to create in test_pose_utils.py:
    - make_rotation_matrix(pitch=0, yaw=0, roll=0, scale=1.0) -> 4x4 ndarray helper
    - Synthetic frontal 3D landmarks with known iris positions (478x3)
    - Use SciPy Rotation to create known rotation matrices for ground truth
  </implementation>
