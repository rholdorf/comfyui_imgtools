<feature>
  <name>Face Model NPZ Persistence</name>
  <files>utils/model_io.py, tests/test_model_io.py</files>
  <behavior>
    - save_face_model(path, canonical_landmarks, head_dimensions, control_indices, landmark_stddev) creates a .facemodel.npz file
    - load_face_model(path) returns dict with version, canonical_landmarks (478x2 float64), head_dimensions (dict with width/height/depth floats), control_indices (1-d int64), landmark_stddev (478x2 float64)
    - Round-trip: save then load produces bit-identical arrays (np.testing.assert_array_equal) and equal head_dimensions dict
    - load_face_model on nonexistent path raises FileNotFoundError
    - load_face_model on file missing required NPZ keys raises ValueError matching "missing fields"
    - load_face_model on file with version != MODEL_VERSION raises ValueError matching "Unsupported model version"
    - load_face_model on file with wrong array shape (e.g. landmarks (100,2) instead of (478,2)) raises ValueError matching "expected shape"
    - Saved file size is under 15 KB (requirement says ~6 KB; allow margin for compression variance)
  </behavior>
  <implementation>
    Follow the exact pattern from 06-RESEARCH.md:

    **Schema (MODEL_VERSION = "1"):**
    - version: 0-d unicode string array
    - canonical_landmarks: float64 (478, 2)
    - head_dimensions: float64 (3,) -- [width, height, depth], reconstructed to dict on load
    - control_indices: int64 (N,) -- variable length
    - landmark_stddev: float64 (478, 2)

    **save_face_model:** Convert head_dimensions dict to (3,) array, cast all arrays to explicit dtypes, call np.savez_compressed.

    **load_face_model:** Check path exists (FileNotFoundError). Use `np.load(path, allow_pickle=False)`. Check all required keys present (ValueError with "missing fields"). Check version string match (ValueError with "Unsupported model version"). Validate each array's dtype kind and shape (ValueError with "expected shape"). Extract all arrays into plain dict before returning (avoids unclosed file handle). Reconstruct head_dimensions from (3,) array back to dict.

    **Important:** Use `allow_pickle=False` explicitly. Store version as `np.array(MODEL_VERSION)` (string, not int). Cast landmarks/stddev to float64 and control_indices to int64 on save.

    Add `utils/model_io.py` to `utils/__init__.py` if __init__.py has explicit imports (check first; if empty, skip).
  </implementation>
</feature>
