  <action>
    Create utils/model_builder.py with these functions:

    1. `SUPPORTED_EXTENSIONS = {".jpg", ".jpeg", ".png", ".webp", ".bmp"}`

    2. `scan_images(directory: str) -> list[Path]` — flat scan of directory, filter by extension, sort by name. Raise ValueError if not a valid directory.

    3. `process_image(image_path: Path, landmarker, yaw_threshold=45.0, pitch_threshold=30.0) -> dict` — loads image via PIL (convert RGB), creates MediaPipe Image, detects face via landmarker, extracts landmarks, checks pose thresholds. Returns dict with keys: filename, status ("ACCEPTED"/"REJECTED"/"NO FACE"), yaw, pitch, roll, confidence, weight, normalized_3d (478,3), head_dims. For pose=None fallback: use landmarks_3d directly with normalize_landmarks_3d (no frontalization), weight=1.0. For confidence: access detection confidence from MediaPipe result if available, else 0.0.

    4. `compute_weighted_average(accepted_data: list[dict]) -> tuple[np.ndarray, np.ndarray]` — weighted mean of normalized_3d arrays using weight field. Returns (mean_3d (478,3), stddev_3d (478,3)). Uses formula from RESEARCH.md.

    5. `build_face_model(directory: str, yaw_threshold=45.0, pitch_threshold=30.0, save_path="") -> tuple[dict, list[dict], str]` — orchestrates: scan_images -> process each -> compute_weighted_average -> project to 2D (drop Z column) -> compute head_dims weighted average -> save via save_face_model -> return (model_dict, per_image_results, actual_save_path). Default save_path: `{directory}/face_model.facemodel.npz`. Raises ValueError if no accepted images.

    - Integration test for build_face_model with mocked MediaPipe detection
