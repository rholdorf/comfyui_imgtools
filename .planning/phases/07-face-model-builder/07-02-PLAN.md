<task type="auto">
  <name>Task 1: Create FaceModelBuilder node with quality report and preview</name>
  <files>face_model_builder.py</files>
  <action>
    Create face_model_builder.py at package root (same level as face_detection.py) with:

    1. **FaceModelBuilder class** following ComfyUI node pattern:
       - INPUT_TYPES:
         - required: directory (STRING, default "")
         - optional: yaw_threshold (FLOAT, default 45.0, min 0, max 90, step 1), pitch_threshold (FLOAT, default 30.0, min 0, max 90, step 1), save_path (STRING, default "")
       - RETURN_TYPES = ("FACE_MODEL", "STRING", "IMAGE")
       - RETURN_NAMES = ("face_model", "quality_report", "preview")
       - FUNCTION = "build_model"
       - CATEGORY = "imgtools/face"

    2. **build_model method**: Calls `build_face_model(directory, yaw_threshold, pitch_threshold, save_path)` from utils/model_builder.py. Then generates quality_report string and preview image from the results. Returns (model_dict, report_string, preview_tensor).

    3. **format_quality_report(results, save_path, yaw_thresh, pitch_thresh) -> str** function:
       - Plain text table with aligned columns: File | Status | Yaw | Pitch | Roll | Conf | Weight
       - Sort order: ACCEPTED first (by weight descending), then REJECTED (by yaw ascending), then NO FACE (by filename)
       - For NO FACE rows: show "N/A" for angle/confidence/weight columns
       - For REJECTED rows: show "-" for weight column
       - Summary line: "Total: N | Accepted: N | Rejected: N | No face: N"
       - Threshold line: "Thresholds: yaw=45.0 deg, pitch=30.0 deg"
       - Last line: "Model saved to: /path/to/model.facemodel.npz"

    4. **render_preview(canonical_2d, control_indices, oval_indices, n_images) -> np.ndarray** function:
       - 512x512 black background canvas using PIL Image.new("RGB")
       - Header text at top: "FaceModel (N images)" in white, using PIL default font
       - Scale the 42 MORPH_CONTROL_INDICES points to fit canvas:
         - Compute bounding box of control points only
         - Add ~15% padding (margin=50px)
         - Scale uniformly to fit, center in canvas
       - Draw FACE_OVAL_INDICES as white connected lines (width=1) forming face contour
       - Draw MORPH_CONTROL_INDICES as green filled circles (radius=3)
       - Return numpy array (512, 512, 3) uint8

    5. **Convert preview to ComfyUI IMAGE tensor**: `torch.from_numpy(preview_np.astype(np.float32) / 255.0).unsqueeze(0)` — shape [1, 512, 512, 3]

    Import MORPH_CONTROL_INDICES from utils.morph_utils and FACE_OVAL_INDICES from utils.face_mask.
