    FaceCropAlign.crop_and_align(self, image, landmarks, face_index=0, padding=0.3, align=True):
      1. Clamp face_index: idx = min(face_index, len(landmarks) - 1)
      2. Get selected face landmarks_px = landmarks[idx]["landmarks"]
      3. Convert image tensor to numpy: img_np = image[0].cpu().numpy().astype(np.float64)  -- shape (H,W,3)
      4. Get img dimensions: h, w = img_np.shape[:2]
      5. If align=True: transform, angle = build_alignment_transform(landmarks_px, w, h); aligned = apply_alignment(img_np, transform)
         If align=False: transform = AffineTransform() (identity); angle = 0.0; aligned = img_np
      6. Compute crop box: x1, y1, x2, y2 = compute_padded_crop_box(landmarks_px, transform, padding, w, h)
      7. Crop: cropped = aligned[y1:y2, x1:x2]
      8. Transform landmarks to crop space for mask: offset landmarks by (-x1, -y1) after transform
      9. Generate mask: mask_np = generate_face_mask(crop_landmarks, crop_h, crop_w) where crop_h=y2-y1, crop_w=x2-x1
      10. Build align_data dict: rotation_angle (float), rotation_center (tuple of floats -- eye midpoint), crop_box (x1,y1,x2,y2), original_size (w,h), transform_matrix (transform.params as 3x3 ndarray)
      11. Convert outputs: cropped_tensor = torch.from_numpy(cropped.astype(np.float32)).unsqueeze(0); mask_tensor = torch.from_numpy(mask_np).unsqueeze(0)
      12. Return (cropped_tensor, align_data, mask_tensor)

    Handle edge case: if crop box has zero width or height (shouldn't happen with valid face, but defensively), return a 1x1 black image and empty mask.

    IMPORTANT: Do NOT use OpenCV. Use skimage only (per PLAT-02).
    IMPORTANT: Follow the conditional import pattern from FaceDetect for consistency.
