    1. compute_morph_warp(source_lms, target_lms, strength, img_shape):
       - Extract control points using MORPH_CONTROL_INDICES from both landmark sets
       - Normalize both by inter-eye distance using normalize_landmarks + compute_eye_centers
       - Linear interpolation: morphed_norm = src_norm + strength * (tgt_norm - src_norm)
       - Denormalize back to source pixel space (using source eye center and source ied)
       - Add boundary anchors (map to themselves) via _get_boundary_anchors
       - Stack: src_with_anchors = np.vstack([src_pts, anchors]), dst_with_anchors = np.vstack([morphed_px, anchors])
       - Create ThinPlateSplineTransform(), call estimate(dst_with_anchors, src_with_anchors)
       - CRITICAL: estimate(dst, src) order -- warp() uses inverse mapping
       - Return the TPS transform object (or None on failure)

       - Return as float32
