- MODEL_URL = "https://storage.googleapis.com/mediapipe-models/face_landmarker/face_landmarker/float16/1/face_landmarker.task"
- Global `_landmarker = None` for cached instance
- `get_landmarker(min_detection_confidence=0.5, min_presence_confidence=0.5)` function:
  - Creates `models/` dir inside extension root if needed
  - Downloads model via urllib.request.urlretrieve if not present (print "[ImgTools] Downloading MediaPipe Face Landmarker model...")
  - Creates FaceLandmarker with `vision.FaceLandmarkerOptions(base_options=BaseOptions(model_asset_path=...), running_mode=vision.RunningMode.IMAGE, num_faces=10, min_face_detection_confidence=min_detection_confidence, min_face_presence_confidence=min_presence_confidence)`
  - Caches in global `_landmarker` and returns it
  - NOTE: If confidence params differ from cached instance, recreate it (compare stored params)
- `comfyui_to_mediapipe(image_tensor)` function:
  - Takes ComfyUI IMAGE tensor [batch, H, W, C] float32 [0,1]
  - Converts first image in batch: `(image_tensor[0].cpu().numpy() * 255).astype(np.uint8)`
  - Returns `mp.Image(image_format=mp.ImageFormat.SRGB, data=img_np)`
- Use `from mediapipe.tasks.python import BaseOptions` and `from mediapipe.tasks.python import vision` (NOT legacy mediapipe.solutions)

Create utils/landmarks.py with:
- `extract_landmarks(result, img_width, img_height)` function:
  - Takes a `FaceLandmarkerResult` and image dimensions
  - Returns list of face dicts, each with:
    - "landmarks": np.array shape (478, 2) of pixel coordinates (lm.x * img_width, lm.y * img_height)
    - "landmarks_3d": np.array shape (478, 3) with (lm.x, lm.y, lm.z) normalized coords
  - Returns empty list if no faces detected
- `draw_landmarks_on_image(img_np, face_landmarks_list, img_width, img_height)` function:
  - Takes uint8 numpy array (H, W, 3), list of MediaPipe NormalizedLandmarkList, dimensions
  - Draws green 2x2 dots at each landmark position on a copy
  - Returns modified numpy array
