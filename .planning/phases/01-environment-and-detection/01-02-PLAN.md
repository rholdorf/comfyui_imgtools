Then create face_detection.py with class FaceDetect:
- INPUT_TYPES: required "image" (IMAGE,), optional "min_detection_confidence" (FLOAT, default 0.5, min 0.0, max 1.0, step 0.05)
- RETURN_TYPES = ("FACE_LANDMARKS", "IMAGE", "INT")
- RETURN_NAMES = ("landmarks", "preview", "face_count")
- FUNCTION = "detect_faces"
- CATEGORY = "imgtools/face"

detect_faces(self, image, min_detection_confidence=0.5) method:
1. Get image dimensions from tensor shape: `_, h, w, _ = image.shape`
2. Convert to MediaPipe: `mp_image = comfyui_to_mediapipe(image)`
3. Get landmarker: `landmarker = get_landmarker(min_detection_confidence=min_detection_confidence)`
4. Detect: `result = landmarker.detect(mp_image)`
5. Extract landmarks: `faces = extract_landmarks(result, w, h)`
6. Create preview:
   - Get uint8 numpy array from tensor: `img_np = (image[0].cpu().numpy() * 255).astype(np.uint8)`
   - If faces detected, draw landmarks: `preview_np = draw_landmarks_on_image(img_np, result.face_landmarks, w, h)`
   - Else use original: `preview_np = img_np.copy()`
   - Convert back to ComfyUI tensor: `preview = torch.from_numpy(preview_np.astype(np.float32) / 255.0).unsqueeze(0)`
7. Return: `(faces, preview, len(faces))`

Import from utils using relative imports:
```python

```
