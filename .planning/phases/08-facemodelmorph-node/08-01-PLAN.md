<feature>
  <name>FaceModelMorph: pose-aware model morphing with TPS warp</name>
  <files>face_model_morph.py, tests/test_face_model_morph.py</files>
  <behavior>
    ## Delta Computation — Pose-Aware Path
    - Given source face with pose data and a FACE_MODEL, computes pixel-space delta at 42 control points
    - Pipeline: frontalize source 3D -> normalize by IPD -> project to 2D -> delta vs model canonical -> scale by source IED -> symmetrize
    - effective_strength = user_strength * cos(radians(yaw)) * cos(radians(pitch))
    - morphed_ctrl = src_ctrl_px + effective_strength * px_delta
    - Test: at strength=1.0 with different model and source shapes, morphed_ctrl != src_ctrl (visible displacement)
    - Test: at strength=0.0, morphed_ctrl == src_ctrl (zero displacement regardless of delta)
    - Test: at yaw=80 degrees, effective_strength is near zero (cos(80) ~ 0.17), pixel displacement is minimal
    - Test: at yaw=0, pitch=0 (frontal), effective_strength equals user_strength exactly

    ## Delta Computation — Procrustes Fallback
    - Given source face with pose=None, uses Procrustes alignment of model to source pixel space
    - Scale model control points by source IED, translate to eye midpoint, then procrustes_align
    - No pose attenuation applied (full user_strength)
    - Test: produces valid morphed_ctrl that differs from src_ctrl at strength=1.0
    - Test: result shape is (42, 2)

    ## Model Symmetrization (MRPH-03)
    - symmetrize=True: canonical_landmarks are symmetrized BEFORE delta computation
    - Uses _MORPH_MIRROR_PAIRS landmark IDs as direct array indices on (478,2) array
    - For each pair: average X-distance from midline, average Y positions
    - For midline points: snap X to midline X
    - Test: after symmetrization, mirror pair X-distances from midline are equal
    - Test: after symmetrization, mirror pair Y values are equal
    - Test: symmetrize=False leaves model unchanged

    ## Pose Attenuation (POSE-04)
    - cos(radians(yaw)) * cos(radians(pitch)) multiplied with user strength
    - No floor — near-profile faces get near-zero morph
    - pose=None -> full strength, no attenuation
    - Test: yaw=0,pitch=0 -> factor=1.0; yaw=45 -> factor~0.707; yaw=89 -> factor~0.017

    ## TPS Warp Assembly
    - Boundary anchors from _get_boundary_anchors(w, h)
    - Near-duplicate removal (same pattern as compute_morph_warp, min dist 1e-3)
    - TPS estimation: tps.estimate(dst_with_anchors, src_with_anchors)
    - If tps.estimate returns False -> passthrough
    - Warp with skimage warp(img, inverse_map=tps, ...)
    - Feathered mask from generate_feathered_mask(morphed_full_lms, h, w)

    ## Head Scale
    - model head_dimensions.width / source head_dimensions.width = head_scale_raw
    - head_scale = 1.0 + effective_strength * (head_scale_raw - 1.0)
    - Stored in out_align_data["head_scale"]
    - Fallback (no pose): compute head dims from raw landmarks_3d with IPD from normalize_landmarks_3d
    - If landmarks_3d all zeros (synthetic): default head_scale = 1.0

    ## Passthrough / Graceful Degradation
    - Empty/missing source_landmarks -> passthrough (source image + ones mask + original align_data)
    - IED < 1e-6 -> passthrough
    - TPS estimation failure -> passthrough
    - Any exception -> passthrough
    - Passthrough returns same (IMAGE, MASK, ALIGN_DATA) tuple structure

    ## Node Class Contract
    - INPUT_TYPES: source_image(IMAGE), face_model(FACE_MODEL), source_landmarks(FACE_LANDMARKS), source_align_data(ALIGN_DATA), strength(FLOAT 0-1 default 0.5 step 0.05). Optional: symmetrize(BOOLEAN default False)
    - RETURN_TYPES = ("IMAGE", "MASK", "ALIGN_DATA") — exactly matches FaceShapeMorph
    - RETURN_NAMES = ("morphed_face", "warp_mask", "align_data")
    - FUNCTION = "morph"
    - CATEGORY = "imgtools/face"
